Model Execution Functions¶

Model Execution

ogusa.execute¶

This module defines the runner() function, which is used to run OG-USA

ogusa.execute.runner(p, time_path=True, client=None)[source]¶

This function runs the OG-USA model, solving for the steady-state and (optionally) the time path equilibrium.

Parameters

  • p (Specifications object) – model parameters

  • time_path (bool) – whether to solve for the time path equilibrium

  • client (Dask client object) – client

Returns

None
